# T009: Create `BulkActionsBar` Component

**Status**: `pending`

## Action
Create

## Business Summary
Floating bar showing when bookmarks are selected with bulk action buttons

## Logic
- Fixed position bar at bottom or top of viewport
- Shows "N selected" count text
- Actions: Select All, Clear Selection, Delete Selected (red)
- Only visible when selectedCount > 0
- Animate in/out when selection changes
- Keyboard accessible for all buttons

## Technical Logic
```tsx
interface BulkActionsBarProps {
  selectedCount: number
  visibleCount: number
  onSelectAll: () => void
  onClearSelection: () => void
  onDeleteSelected: () => void
}

// Fixed: bottom-4 left-1/2 -translate-x-1/2
// Framer Motion: AnimatePresence for slide in/out
// Buttons: Select All (ghost), Clear (ghost), Delete (red ghost)
```

## Testing
Visual check:
- Bar appears when selectedCount > 0
- Bar disappears when selectedCount = 0
- Shows correct count ("3 selected")
- All buttons call correct handlers
- Animation is smooth
- Keyboard accessible (Tab through buttons)
