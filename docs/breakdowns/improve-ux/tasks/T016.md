# T016: Add "Copy URL" Button to BookmarkCard

**Status**: `pending`

## Action
Modify

## Business Summary
Quick action to copy bookmark URL to clipboard

## Logic
- Add ghost button next to Edit/Delete actions
- Click calls `navigator.clipboard.writeText(url)`
- Show toast: "URL copied to clipboard"
- Handle clipboard errors gracefully

## Technical Logic
```tsx
const handleCopyUrl = async () => {
  try {
    await navigator.clipboard.writeText(bookmark.url)
    toast.success('URL copied to clipboard')
  } catch (error) {
    toast.error('Failed to copy URL')
  }
}

<Button variant="ghost" onClick={handleCopyUrl}>
  Copy
</Button>
```
