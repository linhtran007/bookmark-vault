# T014: Add CSS Keyframe Animation for Marquee

**Status**: `pending`

## Action
Modify

## Business Summary
Define marquee animation keyframe in global styles

## Logic
- Add `@keyframes marquee` to globals.css
- Smooth scroll from 0 to -100% translateX
- Duration ~10s, linear timing, infinite iteration
- Add `.animate-marquee` utility class

## Technical Logic
```css
@keyframes marquee {
  0% { transform: translateX(0); }
  100% { transform: translateX(-100%); }
}

.animate-marquee {
  animation: marquee 10s linear infinite;
}
```

## Testing
Visual check:
- Animation is smooth (no jitter)
- Duration feels appropriate (~10s)
- Linear timing (no easing)
- Loops seamlessly
