# T011: Integrate `FilterChips` into `BookmarkListView`

**Status**: `pending`

## Action
Modify

## Business Summary
Show active filters as removable chips below toolbar

## Logic
- Add `FilterChips` component below BookmarkToolbar
- Only show when at least one filter is active (non-default)
- Chips update reactively to filter changes
- Chips use current filter state (searchQuery, selectedTag, sortKey)

## Technical Logic
```tsx
{searchQuery || selectedTag !== 'all' || sortKey !== 'newest' ? (
  <FilterChips
    searchQuery={searchQuery}
    selectedTag={selectedTag}
    sortKey={sortKey}
    onClearSearch={() => setSearchQuery('')}
    onClearTag={() => setSelectedTag('all')}
    onResetSort={() => setSortKey('newest')}
  />
) : null}
```

## Testing
Visual check:
- Chips appear for each active filter
- No chips when all filters are default
- Chips update when filters change
- Clicking X on chip clears that filter
