# T013: Create `MarqueeText` Component

**Status**: `pending`

## Action
Create

## Business Summary
Animate overflowing text on hover with accessibility support

## Logic
- Container clips overflow text (overflow-hidden, whitespace-nowrap)
- On hover, animate scroll only if text overflows
- Respect `prefers-reduced-motion` media query
- Fallback to underline for reduced motion users

## Technical Logic
```tsx
interface MarqueeTextProps {
  children: React.ReactNode
}

<div className="overflow-hidden whitespace-nowrap">
  <span className="
    inline-block
    hover:animate-marquee
    motion-reduce:hover:animate-none
    motion-reduce:hover:underline
  ">
    {children}
  </span>
</div>
```

## Testing
Visual check:
- Short text (no overflow): no animation on hover
- Long text (overflow): marquee animation on hover
- With prefers-reduced-motion: underline instead of animation
- Animation is smooth and loops correctly
