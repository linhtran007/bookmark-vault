# T008: Add Checkbox to `BookmarkCard`

**Status**: `pending`

## Action
Modify

## Business Summary
Enable per-card selection with checkbox for bulk actions

## Logic
- Add checkbox input (visually hidden with custom styled box)
- Checkbox state controlled by `useBookmarkSelection` hook
- Clicking card body toggles checkbox (excluding action buttons)
- Show visual indicator (border/background) when card is selected
- Checkbox keyboard accessible (Tab, Space, Enter)

## Technical Logic
```tsx
<input
  type="checkbox"
  checked={isSelected(bookmark.id)}
  onChange={() => toggle(bookmark.id)}
  className="sr-only peer"
/>
<div className="checkbox-visual" />
// Card wrapper uses peer-checked for selected state
```

## Testing
Visual check:
- Checkbox appears on each card
- Clicking checkbox toggles selection
- Selected cards have visual indicator
- Keyboard navigation works (Tab to checkbox, Space to toggle)
- Action buttons don't trigger checkbox toggle
