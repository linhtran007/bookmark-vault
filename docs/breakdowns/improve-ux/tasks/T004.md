# T004: Create `useComprehensiveClearFilters` Hook

**Status**: `pending`

## Action
Create

## Business Summary
Single hook to clear all filters (search, tag, sort) at once

## Logic
- Accept callbacks for each filter reset function
- Provide clearAllFilters method that executes all callbacks
- Simple composition pattern - no internal state

## Technical Logic
```ts
interface UseComprehensiveClearFiltersProps {
  onClearSearch: () => void
  onClearTag: () => void
  onResetSort: () => void
}

const useComprehensiveClearFilters = ({
  onClearSearch,
  onClearTag,
  onResetSort
}: UseComprehensiveClearFiltersProps) => ({
  clearAllFilters: () => {
    onClearSearch()
    onClearTag()
    onResetSort()
  }
})
```

## Testing
Unit test:
- All callbacks executed when clearAllFilters is called
- No callbacks executed if clearAllFilters not called
