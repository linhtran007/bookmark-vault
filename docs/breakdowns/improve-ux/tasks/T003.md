**Status**: `pending`

## Business Summary
Visual indicator pills showing active filters (search, tag, sort) with quick remove buttons

## Logic
- Show chip for each active filter (non-default values)
- Search chip: shows when searchQuery is not empty
- Tag chip: shows when selectedTag is not "all"
- Sort chip: shows when sortKey is not "newest"
- Each chip has X button to reset that specific filter
- Chips are dismissible via click or keyboard (Enter/Space)

## Technical Logic
```tsx
interface FilterChipsProps {
  searchQuery: string
  selectedTag: string
  sortKey: SortKey
  onClearSearch: () => void
  onClearTag: () => void
  onResetSort: () => void
}

// Render chips horizontally with gap
// Each chip: pill shape, removable X icon
// Click X or press Enter/Space to remove
```

- Each chip shows correct label/value
- Remove button calls correct callback
- All chips render when multiple filters active
