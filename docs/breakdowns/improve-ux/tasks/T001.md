# T001: Create `useBookmarkSelection` Hook

**Status**: `pending`

## Action
Create

## Business Summary
Manages selection state for bulk actions on bookmarks

## Logic
- Track selected bookmark IDs in a Set data structure
- Provide methods: toggle, selectAll, clearSelection, isSelected
- selectAll should only select visible filtered results (not all bookmarks)
- Returns selectedCount for display purposes

## Technical Logic
```ts
interface SelectionState {
  selectedIds: Set<string>
  toggle: (id: string) => void
  selectAll: (visibleIds: string[]) => void
  clearSelection: () => void
  isSelected: (id: string) => boolean
  selectedCount: number
}
```

## Testing
Unit test for all selection methods:
- toggle adds ID to Set if not present, removes if present
- selectAll replaces Set with provided IDs
- clearSelection empties the Set
- isSelected returns correct boolean
- selectedCount returns Set.size
- Edge cases: empty array, single ID, all selected, partial selection
